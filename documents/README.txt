These are the strps to run the code we implemented :-

1) Setup Opengl and Visual studio 2008 or higher version in the system.
   Here is the link given below to setup opengl in visual studio.

     a)  https://www.youtube.com/watch?v=8p76pJsUP44 

2) Install cuda toolkit from the NVIDIA website.

3) Setup cuda in the visual studio.
   Here is the link given to setup cuda.
   
   a) https://www.youtube.com/watch?v=Cm7W6MaNuF4

4) Create an empty console application 
5) Right click on the project and open properties -> Set path to openGL
6) Right Click on the project and add file -> Select "volumeRender.cpp" and "volumeRender_kernel.cu" from the given folder inside code folder
7) Right Click on the project and open the project location -> copy the contents of the data folder to this directory
8) open the cpp file and run it
